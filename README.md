# Relative Assignment
This project is made according to the guildlines provided in the notion document and figma design provided.

## How to setup on a Local machine?
Install all dependecies
```bash
#using npm 
npm i --save
#using yarn 
yarn add
#using pnpm
pnpm i
```
Starting a local Dev Server
```bash
#using npm 
npm run dev
#using yarn 
yarn dev
#using pnpm
pnpm dev
```
